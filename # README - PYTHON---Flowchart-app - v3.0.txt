# 🧠 Flowchart Generator

A tiny Python tool that creates styled flowcharts using `diagrams` and `tkinter`.

## 🔧 Features
- Input title via popup 🖊️  
- Auto-generates a basic flowchart (start, steps, decision, error)  
- Styled nodes and edges 🎨

## ▶️ Usage
Install:
```bash
pip install diagrams
sudo apt install graphviz
```

Run:
```bash
python your_script.py
```
